About This Project

This website compares mobile plans from Croatian providers using publicly available data.

Purpose

Mobile plan information in Croatia is spread across multiple provider websites. This project collects and presents that data in a single, comparable format.

What’s Implemented

  • Data scraping and normalization
  • Backend built with Flask
  • Sortable comparison tables
  • Responsive layout using Bootstrap

Tech Stack

  • Python (Flask)
  • SQL
  • HTML, CSS, JavaScript
  • Bootstrap

Disclaimer

All data is collected from publicly available sources and may change over time.
